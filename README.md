# 🚀 Project Overview

This project focuses on workforce analytics using Excel and Power Pivot to analyze employee trends, overtime patterns, satisfaction metrics, and department-wise workforce insights through an interactive dashboard.

The dashboard was created using a relational dataset with multiple connected tables and includes KPI reporting, interactive slicers, and business insights for workforce analysis.

---

# 🛠️ Tools & Technologies Used

- Microsoft Excel
- Power Pivot
- DAX
- Pivot Tables
- Slicers
- Data Modeling

---

# 📌 Dashboard Features

✅ KPI Cards  
✅ Department-wise Analysis  
✅ Overtime Insights  
✅ Workforce Metrics  
✅ Interactive Filters  
✅ Employee Satisfaction Analysis  
✅ Business Recommendations  

---

# 📊 Key Insights

- Sales department showed higher overtime workload.
- Workforce metrics varied across departments.
- Employee satisfaction trends were analyzed using KPI reporting.
- Interactive filters helped compare workforce patterns dynamically.

---

# 💡 Recommendations

- Reduce overtime pressure in high-workload teams.
- Improve employee engagement initiatives.
- Monitor workforce wellness metrics regularly.
- Focus on improving work-life balance across departments.
